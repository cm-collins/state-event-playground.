# state-event-playground

**state-event-playground** is a hands-on Jetpack Compose learning repo focused on mastering **state** and **event** management in Kotlin.

It contains small, isolated examples that demonstrate:

- **Unidirectional Data Flow (UDF):** `UiState` + `UiEvent` + `UiEffect`
- **ViewModel-driven state** using `StateFlow`
- **One-time effects** (snackbar, navigation, toast) using `SharedFlow`
- **Clean patterns** for async work (loading, success, error), form validation, and list/UI updates

The goal is to start from **basic** examples and move to **advanced** ones, while keeping the code simple, clear, and testable.
